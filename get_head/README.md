# mtcnn-pytorch
pytorch implementation of  face detection algorithm  MTCNN

### Usage MTCNN

Just download the repository and then do this:  

python get_head_box.py -i img/in -o img/out  

Where:  
- -i is input image folder  
- -o is output image folder
