# AOT-GAN Image Inpainting

### Usage
Just download the repository and then do this:  
python src/inpaint.py --dir_image tst/in  --dir_mask tst/msk --outputs tst/out  
Where:  
- --dir_image is input image folder  
- --dir_mask is input mask folder  
- --outputs is output image folder
